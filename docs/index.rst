``visca_over_ip``
=================

``visca_over_ip`` is a Python driver package for Sony PTZ cameras which use the VISCA over IP protocol.
See :doc:`tutorial` to get started.

.. note::
    This package does not allow you to capture video or stills from the camera.
    It only allows you to control things like pan, tilt, zoom, and focus.
